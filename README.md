# GPT-Generative-pretrained-transformer-
In this project i have implemented a character level gpt style transformer from scratch using pytorch.  
The model created learns to predict the next character given the current character using embedding lookup.
## you can download the dataset from this link: -  
https://raw.githubusercontent.com/karpathy/char-rnn/master/data/tinyshakespeare/input.txt  
you can use any other format but for this project i have saved it in csv format.  
## What you can learn: -   
1.) Attention   
2.) Scaled Dot-product attention  
3.) positional encoding(embedding)   
4.) Feed forward Network  
5.) Layer normalization   
6.) softmax  
you can also refer to the research paper "Attention is all you need".
